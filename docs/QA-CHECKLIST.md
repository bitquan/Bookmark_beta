# QA Smoke Checklist

Use this checklist before release to verify core flows:

## Auth
- Sign up with a new email.
- Confirm you can log in after signup.
- Verify /account loads your profile data.

## Feed
- Create a post.
- Like/unlike a post.
- Add a comment and verify it appears.

## Chat
- Open /chat and select a user.
- Send a message and verify it appears.
- Confirm realtime updates appear after a second client sends a message.

## Curriculum
- Create a new item.
- Search for a book and select a result.
- Delete an item.

## Progress
- Add progress for a curriculum item.
- Verify dashboard totals update after refresh.

## Profile
- Open a user profile and verify stats.
- Update profile settings and avatar.
